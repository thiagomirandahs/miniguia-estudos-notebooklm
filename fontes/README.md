# 📚 Fontes do Caderno — IA Generativa e LLMs

Esta pasta documenta a **curadoria** das fontes usadas no NotebookLM e serve como local para
guardar os **PDFs baixados** (opcional).

> ℹ️ Por padrão, o [`.gitignore`](../.gitignore) **ignora PDFs** (`fontes/*.pdf`) para manter o
> repositório leve — todas as fontes são abertas e podem ser rebaixadas pelos links abaixo.
> Se quiser versionar os PDFs, edite o `.gitignore`.

---

## ✅ Critérios de curadoria

Toda fonte deste caderno passou por 4 perguntas:

1. **É aberta/gratuita?** — qualquer pessoa consegue acessar.
2. **Tem autoridade?** — autor/instituição reconhecidos (arXiv, Google, autores citados).
3. **A licença permite uso?** — preferi *Creative Commons* e materiais de acesso aberto.
4. **Complementa as demais?** — equilíbrio entre teoria (papers), didática (artigos) e prática (guias).

## ⬇️ Como baixar os PDFs (F1 e F3)

No arXiv, clique em **"View PDF"** (ou use o link direto abaixo) e salve nesta pasta:

- **F1:** `https://arxiv.org/pdf/1706.03762` → salve como `F1-attention-is-all-you-need.pdf`
- **F3:** `https://arxiv.org/pdf/2303.18223` → salve como `F3-survey-of-llms.pdf`

- Autor/Instituição: <quem>

## ⚠️ Boas práticas de integridade

- **Revalide os links** antes de entregar (sites mudam de URL).
- **Respeite as licenças** — cite o autor e não redistribua material proibido.
- **Prefira a fonte primária** (o paper) à explicação de terceiros quando precisar de precisão.
