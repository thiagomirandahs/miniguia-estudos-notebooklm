# 🖼️ Evidências (prints do NotebookLM)

Use esta pasta para guardar **capturas de tela** que comprovam o seu estudo prático — elas
deixam o portfólio mais convincente e mostram que você **realmente executou** o caderno.

## O que printar

- O **notebook montado** com as 5 fontes carregadas.
- Uma **resposta com citações** (mostrando que a IA ancorou na fonte).
- Um **antes/depois** de prompt (a "cicatriz" — prompt ruim vs. prompt refinado).
- O **Mapa mental** ou o **Áudio Overview** gerado pelo NotebookLM.

## Como referenciar no README

Salve as imagens aqui (ex.: `notebook-com-fontes.png`) e use no README:

```markdown
![Notebook com as 5 fontes carregadas](./assets/notebook-com-fontes.png)
```

> 💡 Nomeie os arquivos de forma descritiva (`caso2-prompt-v1-vs-v2.png`) e prefira `.png`.
